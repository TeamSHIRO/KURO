# License Header Template

## C sources
For .c or .h files:
```c++
/*
 * File: file.c
 * Description: Do something.
 *
 * Copyright (C) 2025 Your Name
 * SPDX-License-Identifier: Apache-2.0
 */
```

## Assembly sources
For .asm and .inc files:
```asm
;
; File: file.asm
; Description: Do something.
;
; Copyright (C) 2025 Your Name
; SPDX-License-Identifier: Apache-2.0
;
```

## Notes
Replace Copyright year, format and name with your own.
